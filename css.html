CSS

CSS (Cascading Style Sheets) é a linguagem usada para definir a aparência e layout das páginas feitas com HTML, constrolando cores, fontes, espaçamentos, tamanho e posicionamento dos elementos; em resumo, enquanto o HTML estrutura o conteúdo, o CSS deixa tudo visualmente organizado e bonito.
